# Prosperity Diffusion: 2025–2100

Interactive real-time simulation modeling the geographic diffusion of innovation-led economic growth and its impact on global poverty reduction.

## Features

- **Global Choropleth Map**: Country-level poverty rates (extreme & moderate) updated annually using D3.js
- **Growth Nodes**: 30+ cities classified as core, regional, or emerging innovation hubs
- **Animated Diffusion Flows**: Trade, FDI, digital connectivity, migration, and knowledge spillovers
- **Simulation Engine**: Endogenous feedback loops, structural transformation, and Kuznets dynamics
- **Metrics Dashboard**: Poverty headcount, Gini coefficient, GDP per capita (PPP), urbanization, employment shifts
- **Scenario Controls**: Baseline, Accelerated Reform, and Fragmentation trajectories
- **Policy Interventions**: Education expansion, broadband rollout, trade agreements, climate shocks, governance reform
- **Regional Layers**: Sub-Saharan Africa, South Asia, Southeast Asia, Latin America
- **Time Slider**: Interactive 2025–2100 timeline with adjustable playback speed

## Data Sources

Calibrated from World Bank PovcalNet, IMF WEO, UN Population Division, and OECD datasets.

## Deployment

Static site — deploy to any CDN or hosting platform (Cloudflare Pages, Netlify, Vercel, GitHub Pages).

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- D3.js v7 + TopoJSON for map visualization
- No build tools required
